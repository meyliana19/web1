Pengertian HTML Dasar

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web.

HTML adalah bahasa yang digunakan untuk mendeskripsikan struktur konten di web. HTML menggunakan "tag" untuk menandai elemen-elemen dalam dokumen. Setiap tag biasanya terdiri dari nama tag yang dibuka dan ditutup, dan dapat memiliki atribut untuk memberikan informasi tambahan.
